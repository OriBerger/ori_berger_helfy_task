# Task Manager

A fullstack app for managing tasks.  
The tasks are stored in-memory on the server(no database).

## How to run

### 1. Start the backend
Open a terminal:
```
bash
cd backend
npm install
npm start
```
Runs on port 4000

### 2. Start the frontend
Open another terminal:

```
bash
cd frontend
npm install
npm start
```
Runs on port 3000
